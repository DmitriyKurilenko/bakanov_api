# Release Notes

Заметки о релизах для конечных пользователей. Язык — русский, без технических деталей.
Новые записи — сверху. Группировка по дате, затем по секциям.

## Секции
- **Новое** — новые функции
- **Улучшения** — доработки существующего
- **Исправления** — багфиксы

---

## 2026-05-14

### Новое
- В Битрикс24 появилась возможность создавать договоры прямо из карточки сделки: откройте вкладку «Договор», проверьте автоматически подтянутые данные, отредактируйте при необходимости и нажмите «Сформировать» — PDF будет загружен в сделку и отправлен на email клиента.

## 2026-05-13

### Улучшения
- Фильтр спам-заявок в Битрикс24 теперь срабатывает автоматически: при переносе лида в этап «СПАМ» Client ID сразу отправляется в Яндекс.Метрику.

### Новое
- В Битрикс24 добавлен фильтр спам-заявок: при получении спам-лида или спам-сделки система автоматически извлекает Client ID и передаёт его в Яндекс.Метрику для анализа источников трафика.

## 2026-04-10

### Новое
- В Битрикс24 появилось встроенное приложение: информация о текущем пользователе, статистика по CRM (лиды, сделки, контакты, компании) и таблица последних лидов — прямо в интерфейсе Битрикса.

## 2026-04-09

### Новое
- Добавлена интеграция с Битрикс24: приём и отправка данных по CRM-событиям (лиды, сделки, контакты, компании).
